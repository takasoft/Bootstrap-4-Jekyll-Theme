{% comment %} Related posts by tag + category overlap. Score = (# of shared tags) + (# of shared categories) with the current page. Sorted by score desc, then date desc (lexicographic on "N|YYYY-MM-DD|..."). Renders nothing if the current page has no tags or categories, or if no other post shares any. Backward-compatible: posts that only use `categories:` still get related links, posts that only use `tags:` also work, and posts that use both get a combined score.
